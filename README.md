# 🌾 Smart Agriculture ML

A **Machine Learning based Smart Agriculture Decision Support System** that helps farmers, researchers, and agricultural experts make **data-driven decisions** for crop cultivation.

The system provides intelligent predictions for:

- 🌱 Crop Recommendation
- 🧪 Fertilizer Recommendation
- 🌾 Crop Yield Prediction

The application is built using **Python, Machine Learning, and Streamlit** with a **bilingual interface (English + Hindi)** to make it accessible for farmers.

---

# 🎯 Project Objective

Agriculture is highly dependent on **soil nutrients, weather conditions, and farming practices**. Many farmers lack access to advanced analytical tools to make informed decisions.

The objective of this project is to build an **AI-powered agricultural assistant** that:

- Recommends the **best crop** based on soil and climate conditions
- Suggests the **appropriate fertilizer**
- Predicts **expected crop yield**
- Provides a **simple interface usable by farmers and experts**
- Supports **Hindi and English language options**

This helps improve **productivity, fertilizer efficiency, and sustainable farming practices**.

---

# 📊 Dataset Used

The project uses agricultural datasets containing soil nutrients, environmental conditions, and crop yield information.

## Crop Recommendation Dataset

Features:
- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH value
- Rainfall

Target:
- Recommended Crop

---

## Fertilizer Recommendation Dataset

Features:
- Soil Nutrient Values
- Crop Type
- Soil Type
- Temperature
- Moisture

Target:
- Fertilizer Type

Example fertilizers:
- Urea
- DAP
- 17-17-17
- 20-20
- 28-28

---

## Yield Prediction Dataset

Features:
- Crop
- Season
- Area
- Production
- Soil Nutrients
- Climate Factors

Target:
- Predicted Crop Yield

---

# 🧠 Machine Learning Models

Different ML models are used for prediction.

| Module | Model Used |
|------|------|
| Crop Recommendation | Random Forest |
| Fertilizer Recommendation | Decision Tree |
| Yield Prediction | Random Forest Regressor |

Models are trained and saved using **Joblib/Pickle**.
